Requirements

What you need

GPU

NVIDIA RTX 3060 or better with 8GB+ VRAM. AMD support coming soon.

Network

100 Mbps+ internet connection. Lower latency is better.

Platform

Windows 10/11, macOS 12+, or Linux. Docker required.

Security

Your machine is protected

We take security seriously. Hosts never run arbitrary user code.

  • Sandboxed Docker containers
  • Only network-approved, signed workloads
  • Outbound-only connections
  • No access to your files or network
  • Resource limits enforced
